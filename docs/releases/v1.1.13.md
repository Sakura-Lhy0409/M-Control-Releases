# M-Control v1.1.13

发布日期：2026-07-15

M-Control v1.1.13 修复软件内“检查更新”在 GitHub API 返回 403、最新 Release 刚发布或网络响应较慢时可能查不到新版本的问题。

## 主要更新

### 更新检查可靠性

- GitHub API 不可用时，继续通过 Releases 最新版本跳转地址兜底。
- 兜底请求禁用本机与代理缓存，并附加一次性查询参数，避免刚发布后仍读到旧版本标签。
- 更新检查的前端等待时间由通用 15 秒调整为独立 30 秒，覆盖 API 失败后继续请求兜底地址的完整耗时。
- 手动点击“检查更新”始终展示真实的新版本，不再被此前保存的“忽略此版本”状态压掉；启动时自动检查仍会尊重忽略设置。
- 日志会记录最终通过 GitHub API 或 Releases 跳转拿到的版本标签，后续诊断不再只能看到 403。

## 发布包

- `MControl_v1.1.13_Setup.exe`：普通安装包，包含系统 WebView2 与 PawnIO 前置条件处理。
- `MControl_v1.1.13_Setup_WebView2_Fixed.exe`：内嵌 Microsoft WebView2 Fixed Version Runtime 的安装包，适合无法正常安装或加载系统 WebView2 的精简系统。
- `MControl_v1.1.13_Portable.zip`：普通免安装便携版，使用系统已安装的 WebView2 Runtime。
- `SHA256SUMS.txt`：三个正式产物的 SHA256 校验清单。

## 升级提示

- 从 `v1.1.0` 或更高版本可直接运行 `MControl_v1.1.13_Setup.exe` 覆盖安装。
- 如果旧版软件内检查更新仍无结果，请从本仓库 Releases 页面手动下载并覆盖安装一次 `v1.1.13`；本次修复只能在安装新版本后接管后续更新检查。
- 早于 `v1.1.0` 的安装版建议先卸载旧版，再安装本版本。
- 便携版必须完整解压后运行；需要内嵌 WebView2 时请直接使用 Fixed 安装包。
