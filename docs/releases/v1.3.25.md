# M-Control v1.3.25

发布日期：2026-07-23

本版本修复 LM / True Color 切换时，部分环境下主进程被 Windows shell 的 native WinEvent 回调带崩的问题，同时减少校色页面在 shell 窗口事件密集时的卡顿。

## 主要更新

- 移除主窗体 `SetWinEventHook` shell 回调链，避免 `IL_STUB_ReversePInvoke` 边界上的未处理异常终止 M-Control。
- 校色 Gamma 的周期重放继续由独立的 display-color guard 后台 runtime 负责，不改变 LM / True Color 的互斥规则、模式数据和 HDR 守卫。
- 避免在 WinForms UI 线程同步执行 shell 事件触发的校色 runtime 请求，改善页面响应。
- 问题库新增 `BUG-20260723-001`，记录 WER、转储、根因和回归边界。

## 发布包

- `MControl_v1.3.25_Setup.exe`：普通安装版，可直接覆盖升级。
- `MControl_v1.3.25_Portable.zip`：普通便携版，完整解压后运行。
- `SHA256SUMS.txt`：正式产物 SHA256 校验清单。

`v1.3.25` 属于 hotfix，不提供内嵌 Fixed Version WebView2 Runtime 的大型安装包。

## 升级提示

- `v1.1.0` 或更高版本可直接运行安装包覆盖升级。
- 便携版不会自动覆盖自身文件；检查到新版本后会打开对应 Release 页面。
- 如果仍能复现校色切换崩溃，请保留 `%LocalAppData%\M-Control\error.log` 和崩溃时间，便于继续区分其他 native callback 路径。
