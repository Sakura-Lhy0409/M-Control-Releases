# M-Control v1.1.0

发布日期：2026-07-15

M-Control v1.1.0 是相较 v1.0.0 的一次完整稳定版更新，重点覆盖界面、硬件兼容、后台占用、灯效、安装升级和多语言体验。正式版仍不包含实验性的 GPU 超频、VF 曲线、GPU 锁频、游戏 FPS 检测或硬件实时监控叠加层。

## 主要更新

### 全新界面与本地化

- 更新 M-Control 产品图标，以主题橙 `#D23D1D`、几何 M、环绕轨道和完整 `M-Control` 字标统一程序、托盘、安装器及发布页面视觉。
- 重绘 TCC 风格主界面、侧栏、仪表盘、性能模式图标和硬件信息图标，统一深色与 `#D23D1D` 主题。
- 优化高分辨率与混合 DPI 下的布局、仪表圆环、卡片间距和 OSD 显示。
- 重做 SteelSeries 逐键灯效平面键盘预览与选中反馈。
- 完善简体中文、繁体中文和英文界面文本。

### 监控、散热与硬件控制

- 完善 CPU / GPU 温度、占用、频率、功耗、电池、风扇及扩展传感器监控。
- 增强 ThermalInfo、PawnIO、MSI WMI、AMD SMU、CPU MSR 和 GPU 遥测回退链路，不同机型按实际能力显示。
- 优化 CPU / GPU 风扇曲线、手动曲线恢复、全速风扇快捷键、校准并发与写后确认。
- 增加 EC 固件信息、Windows 键开关、Win/Fn 互换、触摸板快捷键和受支持机型的 USB 供电共享控制。
- 完善性能模式、Windows 电源计划、EWP 节能模式、GPU 模式、电池健康与充电上限状态同步。

### 键盘灯效与 True Color

- 扩展 SteelSeries HID 免驱路径、Mystic Light、固件内置灯效、逐键灯效、Reactive 和 Idle 灯效兼容性。
- 修正多个 KLC / ALC / Legacy 设备的 PID、键位拓扑、亮度、速度、辅助灯区及物理发送顺序。
- 发布包不捆绑 `ssdevfactory`、`ssps2` 或 `msihid` 等第三方 SteelSeries 驱动。
- 完善 True Color 面板数据生成、ICC / ICM 应用、HDR 冲突保护和后台校色恢复。

### 后台占用与可靠性

- 托盘最小化时暂停不需要的 WebView2、隐藏页面刷新和硬件传感器采样，并按需恢复。
- 合并 GPU 遥测采样、慢速传感器任务和校色轮询，减少重复查询、子进程和后台唤醒。
- 灯效宿主、键盘 Hook、Timer Resolution 和校色守护改为按需运行或复用状态。
- 强化配置原子写入、日志并发、子进程超时、WMI / Timer / native handle 释放和退出清理。
- 伴随诊断工具使用独立私有依赖目录，避免 DLL 相互覆盖。

### 安装、升级与发布

- `v1.1.0` 建立后续覆盖升级基线；早于 `v1.1.0` 的版本建议先卸载旧版，再安装本版本。
- 发布流程增加事务式 staging、失败回滚、文件清单、禁入文件检查和 SHA256 校验。
- 本次提供三个程序包：
  - `MControl_v1.1.0_Setup.exe`：普通安装包，适合大多数用户。
  - `MControl_v1.1.0_Portable.zip`：普通免安装便携版。
  - `MControl_v1.1.0_Portable_WebView2_Offline.zip`：在普通便携版基础上附带微软官方 x64 WebView2 离线安装程序，适合缺少 WebView2 的精简系统。
- WebView2 离线便携版不会自动安装 Runtime；完整解压后先运行包内安装程序，再运行 `MControl.exe`。

## 修复与兼容改进

- 修复仪表刷新、隐藏页面 DOM 更新、托盘退出、GPU 状态缓存和后台任务生命周期中的多项竞态。
- 修复不同机型风扇百分比、静音模式读回、场景模式同步、电池信息和 GPU 功耗显示的兼容问题。
- 修复 SteelSeries 灯效路由、固件效果、亮度档位、退出保持和恢复时序问题。
- 修复安装覆盖升级时的旧进程、受管文件和依赖清理问题。

## 升级提示

- 早于 `v1.1.0` 的安装版建议先卸载旧版，再运行 `MControl_v1.1.0_Setup.exe`；以后从 `v1.1.0` 升级到新版本时可直接覆盖安装。
- 便携版用户请完整解压到新目录，不要在 ZIP 内直接运行。
- 若与 MSI Center、Dragon Center 或其他硬件控制工具同时运行，可能发生控制冲突。
- 不同 MSI 机型、BIOS 和 EC 固件差异较大；首次使用新的硬件控制项时请逐项确认读回结果。
