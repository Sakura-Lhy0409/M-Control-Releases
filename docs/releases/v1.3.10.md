# M-Control v1.3.10

发布日期：2026-07-17

本版本把四分区键盘灯效列表与 MSI Center 对齐，补齐官方 `Customize / 自定义`。它不是单一固件效果，而是四个区域分别保存并写入自己的灯效、颜色、色板和速度。

## 主要更新

- 四分区灯效列表新增“自定义”，顺序与 MSI Center 保持一致。
- 四个区域可分别选择关闭、纯色、呼吸或色彩循环。
- 呼吸每区支持 2-4 色，色彩循环每区支持 2-7 色。
- 每区呼吸/色彩循环周期支持 3.0-12.0 秒，并按 MSI 官方的 0.1 秒单位写入。
- 新增四区专用编辑器，不再把四分区机型错误渲染成 SteelSeries 全键盘逐键布局。
- 自定义配置随已知灯效状态和本地灯效配置保存，旧版 `customMap` 静态颜色请求仍可兼容。
- 后端严格按一区到四区执行“选区 + 效果”Feature Report，四区全部成功后只保存一次。

## 发布包

- `MControl_v1.3.10_Setup.exe`：普通安装版，可直接覆盖升级。
- `MControl_v1.3.10_Portable.zip`：普通便携版，完整解压后运行。
- `SHA256SUMS.txt`：正式产物 SHA256 校验清单。

`v1.3.10` 属于 hotfix，不提供内嵌 Fixed Version WebView2 Runtime 的大型安装包。

## 验证结果

- 118 项 Node 静态与行为测试全部通过。
- 193 项 .NET 测试全部通过。
- Release 构建 0 警告、0 错误。
- 字节级测试锁定四区自定义关闭、纯色、呼吸和色彩循环的官方字段、色点与 3-12 秒周期。
- 浏览器验证覆盖桌面与 720px 窄窗口，四区编辑器无横向溢出。
- 当前没有四分区实机，仍需复验四区混合效果一次应用后的实际表现。

## 升级提示

- `v1.1.0` 或更高版本可直接运行安装包覆盖升级。
- 四分区键盘用户可在灯效列表末尾选择“自定义”，分别编辑四个区域。
- 便携版不会自动覆盖自身文件，检查到新版本后会打开对应 Release 页面。
