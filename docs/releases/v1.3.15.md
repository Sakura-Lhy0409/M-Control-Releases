# M-Control v1.3.15

发布日期：2026-07-18

本版本优化仪表盘硬件采样：LibreHardwareMonitor 不再随可见仪表盘固定扫描，而是降级为 CPU 遥测和 PCH 温度的按需兜底。

## 主要更新

- CPU 功耗和频率优先使用 ThermalInfo、EC/MSR、IPF 与 Windows 性能计数器等现有来源。
- 只有主要来源仍无法得到 CPU 功耗或频率时，才启动 LibreHardwareMonitor。
- PCH 温度仅在扩展传感器视图可见，并且 EC `0x9E` 与 MSI ThermalInfo 主板温度都不可用时，才调用 LibreHardwareMonitor。
- 设置、风扇、灯效等非仪表盘页面不会因为缺少 CPU 遥测而后台加载 LibreHardwareMonitor。
- LibreHardwareMonitor 兜底快照缓存时间由 750 ms 提高到 5 秒，减少重复扫描 CPU 和主板传感器树。
- LibreHardwareMonitor 依赖和兜底能力继续保留，不影响缺少主要遥测来源的少数机型。

## 发布包

- `MControl_v1.3.15_Setup.exe`：普通安装版，可直接覆盖升级。
