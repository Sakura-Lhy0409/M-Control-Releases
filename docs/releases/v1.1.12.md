# M-Control v1.1.12

发布日期：2026-07-15

M-Control v1.1.12 修复 Raider 18 系列 SteelSeries 灯效控制，并调整设置页“界面与反馈”卡片布局。本版本包含未单独发布到 Releases 的 v1.1.11 修复，可由 v1.1.10 或 v1.1.11 直接升级。

## 主要更新

### Raider 18 灯效修复

- 修正 `PID_205A` 高级 KLC 通道的实时与逐键报文头，不再误用物理 KLC 五字节格式。
- 从固件持久灯效切回实时写入前，分别为 `PID_205A` KLC 和 `PID_1161` ALC 执行正确的模式复位。
- 模式复位按 HID 路径缓存，避免 33 ms 动画帧循环重复复位。
- `PID_1161` ALC 局部写入使用真实灯区数量，不再固定声明四个区域。
- 修复已经过 Raider 18 实机验证，键盘与 ALC 灯区恢复正常。

### 设置页布局

- “启动与窗口”和“界面与反馈”在左列独立堆叠。
- “设备控制”和“显示与电源”继续在右列独立堆叠。
- 窄窗口恢复为自然单列顺序，卡片不再掉到设置页底部形成孤立区域。

## 发布包

- `MControl_v1.1.12_Setup.exe`：普通安装包，包含系统 WebView2 与 PawnIO 前置条件处理。
- `MControl_v1.1.12_Setup_WebView2_Fixed.exe`：内嵌 Microsoft WebView2 Fixed Version Runtime 的安装包，适合无法正常安装或加载系统 WebView2 的精简系统。
- `MControl_v1.1.12_Portable.zip`：普通免安装便携版，使用系统已安装的 WebView2 Runtime。
- `SHA256SUMS.txt`：三个正式产物的 SHA256 校验清单。

## 升级提示

- 从 `v1.1.0` 或更高版本可直接运行 `MControl_v1.1.12_Setup.exe` 覆盖安装。
- 早于 `v1.1.0` 的安装版建议先卸载旧版，再安装本版本。
- 便携版必须完整解压后运行；需要内嵌 WebView2 时请直接使用 Fixed 安装包。
- 与 SteelSeries GG、MSI Center、Dragon Center 或其他硬件控制工具同时运行时，可能发生控制冲突。
