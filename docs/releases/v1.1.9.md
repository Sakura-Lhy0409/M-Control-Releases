# M-Control v1.1.9

发布日期：2026-07-15

M-Control v1.1.9 增加便携版 PawnIO 自助修复，并把仪表盘数值跳动改为默认关闭、可由用户自行开启。安装版继续沿用安装前 PawnIO 前置检查，便携版则在缺失依赖时提供明确、可恢复的处理入口。

## 主要更新

### PawnIO 自助修复

- 便携版现在随包附带官方签名的 PawnIO 2.2.0 安装程序。
- “设置 → 硬件诊断”会同时检查 PawnIO 服务注册状态与 `\\.\PawnIO` 设备通道；依赖正常时不显示额外提示。
- 缺失时显示“安装 PawnIO”和“重新检测”，安装通过非静默 UAC 界面完成，不会在后台强制安装驱动。
- 启动安装程序前会校验固定 SHA256；发布脚本还会验证代码签名、产品名和版本，复制后再次核对哈希。
- 正确处理成功、需要重启、已触发重启、用户取消和安装失败状态。失败不会阻止 M-Control 启动，只会让依赖 PawnIO 的功能保持不可用。
- 安装版继续使用原有前置事务，不会把便携版修复程序重复复制到应用目录。

### 仪表盘动效

- 增加“仪表数值跳动”开关，默认关闭。
- 关闭时数值更新方式与 TUXEDO Control Center 的稳定显示接近；需要反馈感的用户可以在设置中重新开启。
- 仪表圆弧保持现有平滑更新，不受该开关影响。

## 发布包

- `MControl_v1.1.9_Setup.exe`：推荐使用的安装包，安装前自动检查并处理 PawnIO 前置条件。
- `MControl_v1.1.9_Portable.zip`：免安装便携版，内含 `Dependencies\PawnIO_setup.exe`，缺失驱动时可从硬件诊断卡自助安装。

## 升级提示

- 从 `v1.1.0` 或更高版本可直接运行 `MControl_v1.1.9_Setup.exe` 覆盖安装。
- 早于 `v1.1.0` 的安装版建议先卸载旧版，再安装本版本。
- 便携版用户请完整解压到新目录，不要在 ZIP 内直接运行；否则依赖安装程序无法被可靠定位。
- 与 MSI Center、Dragon Center 或其他硬件控制工具同时运行时，可能发生控制冲突。
