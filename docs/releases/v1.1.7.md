# M-Control v1.1.7

发布日期：2026-07-15

M-Control v1.1.7 是基于 v1.1.0 覆盖升级基线的维护版本，重点修复安装版 PawnIO 前置条件、灯效关闭状态、CPU 功耗不可用提示和退出阶段日志，并更新 Windows 图标与定制系统性能模式支持。

## 主要更新

### 安装与依赖

- 安装器现在会验证 PawnIO 服务和 `\\.\PawnIO` 设备是否真正可用，而不再只依赖安装程序的表面退出结果。
- 正确处理 PawnIO 安装返回的成功、需要重启和已触发重启状态；安装失败时会中止 M-Control 安装并显示有效错误。
- 修复安装结束后错误提示引用已删除临时文件的问题。

### 性能模式与定制系统

- 增加带有效签名的定制系统身份校验，避免仅凭可伪造文本标记开放专用功能。
- 狂暴模式继续优先采用官方机型能力判断；受支持的定制系统可启用对应分支。
- 定制系统分支使用 C4 场景、Windows“最佳性能”电源模式和“平衡”电源计划，以匹配对应 DTT 策略。

### 灯效、监控与退出

- 修复 SteelSeries 灯效设为关闭后，退出或重新进入页面时被错误恢复为其他效果的问题。
- CPU 功耗无法取得可信读数时显示 `N/A`，并区分 PawnIO 缺失和当前平台无法读取两类原因。
- 修复托盘隐藏或退出过程中访问已释放 WebView2 控件产生的无效错误日志。

### 图标与高 DPI

- Windows 桌面、任务栏、托盘、通知、程序文件和安装器统一使用白底、黑色 M、`#D23D1D` 圆弧的图标版本，避免浅色系统表面丢失白色图形。
- 修复高 DPI 与混合缩放环境下托盘通知布局和仪表盘风扇图标偏移。
- 更新狂暴模式火箭图标。

## 发布包

- `MControl_v1.1.7_Setup.exe`：推荐使用的安装包，会处理 PawnIO 前置条件。
- `MControl_v1.1.7_Portable.zip`：免安装便携版；不会自动安装 PawnIO，依赖 CPU MSR / EC 的功能需自行准备驱动。

## 验证结果

- 91 项 Node 静态与行为测试全部通过。
- 76 项 .NET 测试全部通过。
- Release 构建 0 警告、0 错误。
- Inno Setup 安装包已完成真实编译。

## 升级提示

- 从 `v1.1.0` 或更高版本可直接运行 `MControl_v1.1.7_Setup.exe` 覆盖安装。
- 早于 `v1.1.0` 的安装版建议先卸载旧版，再安装本版本。
- 便携版用户请完整解压到新目录，不要在 ZIP 内直接运行。
- 与 MSI Center、Dragon Center 或其他硬件控制工具同时运行时，可能发生控制冲突。
