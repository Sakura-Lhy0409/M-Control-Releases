# M-Control v1.1.10

发布日期：2026-07-15

M-Control v1.1.10 修复软件内“检查更新”可能一直无结果的问题，并新增真正内嵌 Microsoft WebView2 Fixed Version Runtime 的便携版。安装版和普通便携版继续保留，用户可以按系统环境选择，不需要为了 Fixed Runtime 改用单一发行形式。

## 主要更新

### 检查更新修复

- 更新检查不再由 WebView 前端无限期直连 GitHub API，改为后台原生网络请求。
- 请求显式使用 Windows 系统代理、TLS 1.2、GitHub API 请求头和有限超时。
- GitHub API 不可用时，会通过官方 `releases/latest` 重定向继续确认最新版本。
- 已在真实网络环境中分别验证 API 主路径与发布页重定向兜底路径。
- 版本判断改为逐段数字比较；只有远端版本严格高于本地版本时才提示更新，不会把旧版误报为可更新版本。

### WebView2 Fixed 便携版

- 新增 `MControl_v1.1.10_Portable_WebView2_Fixed.zip`。
- 包内直接包含 Microsoft WebView2 Fixed Version Runtime x64 `150.0.4078.65`，无需安装系统 Evergreen Runtime。
- M-Control 检测到同目录下完整的 `WebView2FixedRuntime` 后直接加载；目录缺失或不完整时仍回退到系统已安装的 Evergreen Runtime。
- 发布脚本固定校验官方 CAB SHA256、微软代码签名、Runtime 版本和核心文件，复制进 ZIP 后再次验证文件数量与体积。
- Fixed Runtime 内容为 260 个文件、约 646.89 MB；压缩后的完整 M-Control Fixed 便携包约 300.72 MB。
- 安装版和普通便携版保持原行为，不会被强制塞入 Fixed Runtime。

## 发布包

- `MControl_v1.1.10_Setup.exe`：推荐使用的安装包，继续处理 WebView2 与 PawnIO 前置条件。
- `MControl_v1.1.10_Portable.zip`：普通免安装便携版，使用系统已安装的 WebView2 Runtime。
- `MControl_v1.1.10_Portable_WebView2_Fixed.zip`：真正内嵌固定版 Runtime，适合无法安装或无法正常加载系统 WebView2 的精简系统。
- `SHA256SUMS.txt`：以上三个产物的 SHA256 校验清单。

## 升级提示

- 从 `v1.1.0` 或更高版本可直接运行 `MControl_v1.1.10_Setup.exe` 覆盖安装。
- 早于 `v1.1.0` 的安装版建议先卸载旧版，再安装本版本。
- 两种便携版都必须完整解压后运行；Fixed 版必须保持 `WebView2FixedRuntime` 与 `MControl.exe` 位于同一目录。
- 与 MSI Center、Dragon Center 或其他硬件控制工具同时运行时，可能发生控制冲突。
