# M-Control v1.3.13

发布日期：2026-07-17

本版本继续优化 Windows 外壳图标的白色底板轮廓：进一步增大圆角半径，使桌面、任务栏和安装器图标更圆润，同时保持黑色 M、`#D23D1D` 圆弧及其几何比例不变。

## 主要更新

- Windows 圆角白底板的圆角半径由约 18% 提高到约 30%，减少偏硬的方形观感。
- Logo 内容继续保持 84% 等比缩放，不改变 M 与圆弧的大小、位置和最近点距离。
- 桌面快捷方式、任务栏、托盘、通知、程序文件和安装器继续共用同一套 Windows 图标资产。
- ICO 保留 16、20、24、32、48、64、128 和 256 像素帧。
- Web 页面、favicon 和关于页继续使用透明 Logo，不加入白色底板。
- 图标门禁锁定新的圆角比例，防止后续重新生成时回退。

## 发布包

- `MControl_v1.3.13_Setup.exe`：普通安装版，可直接覆盖升级。
- `MControl_v1.3.13_Portable.zip`：普通便携版，完整解压后运行。
- `SHA256SUMS.txt`：正式产物 SHA256 校验清单。

`v1.3.13` 属于 hotfix，不提供内嵌 Fixed Version WebView2 Runtime 的大型安装包。

## 升级提示

- `v1.1.0` 或更高版本可直接运行安装包覆盖升级。
- Windows 可能短时间保留旧快捷方式或任务栏图标缓存；重新固定任务栏图标或重启资源管理器后会显示新图标。
- 便携版不会自动覆盖自身文件，检查到新版本后会打开对应 Release 页面。
