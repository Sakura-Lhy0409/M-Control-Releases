# M-Control v1.3.20

发布日期：2026-07-20

本版本补齐 SteelSeries 免驱基础路线的“退出时保持灯效”能力。此前 FW-basic 已能运行棱镜等软件实时效果，但后端仍按设备类型整体禁用设置开关，导致动态灯效无法在退出可见控制台后继续运行。

## 主要更新

- FW-basic 所有标为“（实时）”的效果均可勾选“退出时保持灯效”。
- 棱镜、响应层和闲置层等运行时效果会在退出后保留 LightingHost。
- 关闭、纯色和逐键静态无需后台进程，退出后由硬件保持当前写入。
- 静态实时效果完整断电后，仍会在 M-Control 下次启动时自动恢复。
- 所有“（固件）”效果继续禁用退出保持，避免为 MCU 持久效果保留无意义的后台进程。
- 节能模式关灯守卫和 SteelSeries native Prism 路线保持不变。

## 发布包

- `MControl_v1.3.20_Setup.exe`：普通安装版，可直接覆盖升级。
- `MControl_v1.3.20_Portable.zip`：普通便携版，完整解压后运行。

`v1.3.20` 属于 hotfix，不提供内嵌 Fixed Version WebView2 Runtime 的大型安装包。

## 升级提示

- `v1.1.0` 或更高版本可直接运行安装包覆盖升级。
- 使用 FW-basic 路线时，先应用实时灯效，再到设置页启用“退出时保持灯效”。
- 勾选“随 Windows 自动启动”后，静态实时效果会在登录时恢复；未勾选时会在手动启动 M-Control 后恢复。
- 便携版不会自动覆盖自身文件，检查到新版本后会打开对应 Release 页面。
