# M-Control v1.3.4

发布日期：2026-07-17

本版本修复少数机器首次启动时 WebView2 界面初始化失败、重新打开后又能恢复的问题，并补充安装目录完整性诊断。

## 主要更新

- WebView2 初始化遇到已确认具有瞬态特征的 `0x80070490` 或 15 秒超时时，会等待上一条初始化会话退出、重建界面控件并自动重试一次。
- 缺少 `WebView2Loader.dll`、程序集文件缺失或版本清单不匹配仍作为安装损坏直接报错，不使用重试掩盖真实问题。
- 诊断包新增 `installation_integrity.txt`，记录 M-Control 关键依赖的存在性、受管文件清单、文件版本、程序集版本、SHA256 和已加载程序集位置。
- 新增 WebView2 错误分类单元测试，锁定“瞬态错误可重试、依赖损坏不可重试”的边界。

## 发布包

- `MControl_v1.3.4_Setup.exe`：普通安装版，可直接覆盖升级。
- `MControl_v1.3.4_Portable.zip`：普通便携版，完整解压后运行。
- `SHA256SUMS.txt`：正式产物 SHA256 校验清单。

`v1.3.4` 属于 hotfix，不提供内嵌 Fixed Version WebView2 Runtime 的大型安装包。

## 验证结果

- 115 项 Node 静态与行为测试全部通过。
- 178 项 .NET 测试全部通过。
- Release 构建 0 警告、0 错误。
- 发布目录受管清单包含 WebView2、`System.Memory`、`System.Text.Json` 和主程序配置文件，安装完整性报告已在正式发布目录离线验证。

## 升级提示

- `v1.1.0` 或更高版本可直接运行安装包覆盖升级。
- 便携版不会自动覆盖自身文件，检查到新版本后会打开对应 Release 页面。
- 如果升级后仍提示缺少 DLL 或程序集版本冲突，请覆盖安装后重新生成诊断包；新版诊断包会包含 `installation_integrity.txt`。
