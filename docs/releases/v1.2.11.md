# M-Control v1.2.11

发布日期：2026-07-16

本版本汇总 `v1.2.0` 之后的灯效吞吐、风扇曲线恢复、更新检查与首次加载体验修复。

## 主要更新

- 修复 Raider A18 HX 等 PRISM 机型实时灯效帧率过低及错误物理报文路由导致键盘乱亮的问题；KLC 与 ALC 改为独立设备节拍，并保留现场 descriptor 生成边界。
- 修复已保存并应用的手动风扇曲线在程序启动后没有可靠恢复的问题。
- 更新检查改为读取长期固定清单，安装版支持经过大小和 SHA256 校验的下载安装流程，便携版会打开对应版本页面。
- 下载更新时显示实时进度、已下载大小和传输速度。
- GPU 风扇曲线和 GPU 升降档温度输入上限统一为 87°C，CPU 限制不受影响。
- 首次读取风扇曲线前不再展示容易误认的默认预设，改为整块模糊的 `? -> ! -> 实际数据` 状态。
- 优化首次安装后的启动界面，使其颜色、字号和产品主题保持一致。
- 修复手动点击“检查更新”时弹窗全透明，导致检查过程和失败提示都不可见的问题。

## 发布包

- `MControl_v1.2.11_Setup.exe`：普通安装版。
- `MControl_v1.2.11_Portable.zip`：普通便携版，完整解压后运行。
- `SHA256SUMS.txt`：正式产物 SHA256 校验清单。

`v1.2.11` 属于 hotfix，不提供内嵌 Fixed Version WebView2 Runtime 的大型安装包。

- 132 项 .NET 测试全部通过。
