# M-Control v1.3.12

发布日期：2026-07-17

本版本重新处理 Windows 外壳图标：原来的整块纯白正方形改为透明画布上的圆角白底板，保留黑色 M、`#D23D1D` 圆弧及两者已经确认的几何比例。

## 主要更新

- 桌面快捷方式、任务栏、托盘、通知、程序文件和安装器统一使用圆角白底板图标。
- 白色底板外增加透明安全区，消除资源管理器中的硬直方形轮廓。
- 圆弧与 M 整体等比内缩到 84%，不改变二者的相对比例和最近点距离。
- ICO 继续包含 16、20、24、32、48、64、128 和 256 像素帧。
- Web 页面、favicon 和关于页继续使用原来的透明 Logo，不加入白色底板。
- 图标门禁已更新为“透明四角 + 圆角白底板”契约，防止后续生成脚本重新铺满白色画布。

## 发布包

- `MControl_v1.3.12_Setup.exe`：普通安装版，可直接覆盖升级。
- `MControl_v1.3.12_Portable.zip`：普通便携版，完整解压后运行。
- `SHA256SUMS.txt`：正式产物 SHA256 校验清单。

`v1.3.12` 属于 hotfix，不提供内嵌 Fixed Version WebView2 Runtime 的大型安装包。

## 验证结果

- 118 项 Node 静态与行为测试全部通过。
- 196 项 .NET 测试全部通过。
- Release 构建 0 警告、0 错误。
- 16-256 像素 ICO 帧均包含可见黑色 M 和主题色圆弧，四角 alpha 为 0。
- 从最终 `MControl.exe` 和安装器提取的 32px 图标内容一致，圆角透明区域有效。

## 升级提示

- `v1.1.0` 或更高版本可直接运行安装包覆盖升级。
- Windows 可能短时间保留旧快捷方式或任务栏图标缓存；重新固定任务栏图标或重启资源管理器后会显示新图标。
- 便携版不会自动覆盖自身文件，检查到新版本后会打开对应 Release 页面。
